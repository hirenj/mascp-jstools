MASCP Javascript tools

A set of utility Javascript widgets that allows you to render proteomic data in an extensible way. There are three main pieces of functionality:
  • An AJAX data requester - that allows you to retrieve data from various MASCP data sources
  • Libraries to render protein sequences and mark them up with various annotations
  • Map functionality that allows you to mark up areas upon SVG maps

Contact

Hiren Joshi (hjjoshi@lbl.gov)

Compatibility

These libraries are compatible with Mozilla Firefox 3.0 and above, Safari, Chrome and Internet Explorer 7+. For Internet Explorer, the track renderer does not work in all cases, so a text-based renderer is recommended.
